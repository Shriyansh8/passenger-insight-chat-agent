# Passenger Insight Chat Agent

Passenger Insight Chat Agent is an interactive data analytics application built using Python and Streamlit. The application allows users to explore insights from the Titanic passenger dataset through natural language queries and receive both textual responses and visual representations.

Live Application:  
https://passenger-insight-chat-agent-az8ypjdgehamtqejha3ybf.streamlit.app/

---

## Project Overview

This application enables users to:

- Ask dataset-related questions in natural language
- Receive calculated statistical insights
- View automatically generated visualizations
- Interact through a chat-based user interface

The project demonstrates data analysis, UI development, and cloud deployment using Streamlit.

---

## Features

- Chat-style interactive interface
- Rule-based natural language query handling
- Dynamic data visualizations (histograms and bar charts)
- Real-time statistical calculations
- Persistent chat session during interaction
- Public deployment on Streamlit Cloud

- What percentage of passengers were male?
- Show histogram of age

## How to Run Locally

1. Clone the repository

   git clone https://github.com/shriyansh8/passenger-insight-chat-agent.git

2. Navigate into the project directory

   cd passenger-insight-chat-agent

3. Install dependencies

   pip install -r requirements.txt

4. Run the application

   streamlit run app.py
